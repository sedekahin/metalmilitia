== Description ==

This Plugins that makes your site SEO boosting by making your defaut description change everytime you add new post on homepage only. This plugins usefull for those running news site or you are an up to date blogger.
Many others SEO Plugins doesnt have this feature, they just make keep static description and makes google bored if you have an up to date blog.

= Features =

1. Automatically generate description on your homepage based on last 3 title post, this is real good for your site SEO.

== Installation ==

1. Upload `the indowp-agmd.zip file` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! this plugins run automatically, just checked on your homepage source file and find the metatag description.
